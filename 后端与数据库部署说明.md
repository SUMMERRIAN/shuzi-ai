# 树子AI后端与 PostgreSQL 部署说明

当前后端包含：

- PostgreSQL 学生档案数据库
- 注册 / 登录 / 当前账号状态
- 会员状态、存储额度、LT余额
- 会员开通申请、LT充值申请
- 管理员手动开通会员、手动充值LT
- 学情问卷、学情陈述写入学生档案
- 试卷/错题图片AI分析
- 语音陈述转写
- AI相似题生成
- AI知识图生成

## 1. 服务器安装 PostgreSQL

```bash
sudo apt update
sudo apt install -y postgresql postgresql-contrib
sudo systemctl enable postgresql
sudo systemctl start postgresql
```

创建数据库和账号：

```bash
sudo -u postgres psql <<'SQL'
CREATE USER shuzi_ai WITH PASSWORD '请改成强密码';
CREATE DATABASE shuzi_ai OWNER shuzi_ai;
GRANT ALL PRIVILEGES ON DATABASE shuzi_ai TO shuzi_ai;
SQL
```

## 2. 配置后端环境变量

```bash
cd /var/www/shuzi-ai
sudo tee .env >/dev/null <<'EOF'
DATABASE_URL=postgres://shuzi_ai:请改成强密码@127.0.0.1:5432/shuzi_ai
JWT_SECRET=请改成一串很长的随机字符串
ADMIN_SETUP_TOKEN=请改成管理员操作令牌
PORT=3001
CORS_ORIGIN=https://xyshuziai.com
OPENAI_API_KEY=这里填写服务器专用OpenAI密钥
OPENAI_MODEL_TEXT=gpt-5
OPENAI_MODEL_IMAGE=gpt-5
OPENAI_MODEL_TRANSCRIBE=gpt-4o-mini-transcribe
UPLOAD_DIR=/var/www/shuzi-ai/uploads
MAX_UPLOAD_MB=25
EOF
```

## 3. 启动 API 服务

先测试：

```bash
cd /var/www/shuzi-ai
npm install
npm run server
```

看到 `Shuzi AI API listening on 3001` 后，按 `Ctrl+C` 停止测试。

创建 systemd 服务：

```bash
sudo tee /etc/systemd/system/shuzi-ai-api.service >/dev/null <<'EOF'
[Unit]
Description=Shuzi AI API
After=network.target postgresql.service

[Service]
Type=simple
WorkingDirectory=/var/www/shuzi-ai
EnvironmentFile=/var/www/shuzi-ai/.env
ExecStart=/usr/bin/npm run server
Restart=always
RestartSec=5
User=ubuntu

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl daemon-reload
sudo systemctl enable shuzi-ai-api
sudo systemctl restart shuzi-ai-api
sudo systemctl status shuzi-ai-api --no-pager -l
```

## 4. 配置 Nginx 代理 API

在 `server` 配置里加入 `/api`：

```nginx
location /api/ {
    proxy_pass http://127.0.0.1:3001/api/;
    proxy_http_version 1.1;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
}
```

然后：

```bash
sudo nginx -t
sudo systemctl reload nginx
```

## 5. 检查接口

```bash
curl https://xyshuziai.com/api/health
```

正常返回：

```json
{"ok":true,"service":"shuzi-ai-api"}
```

## 6. 管理员手动开通会员

```bash
curl -X POST https://xyshuziai.com/api/admin/memberships/activate \
  -H "Content-Type: application/json" \
  -H "x-admin-token: 你的ADMIN_SETUP_TOKEN" \
  -d '{"identifier":"学生邮箱或手机号","planId":"monthly"}'
```

## 7. 管理员手动充值 LT

```bash
curl -X POST https://xyshuziai.com/api/admin/lt/recharge \
  -H "Content-Type: application/json" \
  -H "x-admin-token: 你的ADMIN_SETUP_TOKEN" \
  -d '{"identifier":"学生邮箱或手机号","packageId":"lt-990"}'
```
